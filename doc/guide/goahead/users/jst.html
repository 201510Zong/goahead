Javascript Templates (JST)

Javascript Templates (JST) is embeds Javascript code in HTML pages to serve dynamic content. An JST document has an ".jst" extension and uses embedded scripting to insert dynamic data into the page before it is sent to the user's browser.

engines can be selected at run time. Individual pages can use multiple scripting engines if required.

To create an JST script field in a JST document, use the <% and %> delimiters. For example, the following JavaScript will output "Hello World" in place of the JST-delimited field:

Today is <% write("Hello World"); %>

Scripting Execution

When a user's browser requests an JST document, the JST handler is invoked to serve the request. The JST document is read from the file system or ROM store in a one-pass operation. Text before the JST delimiters is copied directly to the requesting browser. If JST delimiters are found, the text between the delimiters is passed to the relevant scripting engine for execution. The resulting text is passed back to the browser, and the process continues until the end of the document.
