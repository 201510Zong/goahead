Embedthis GoAhead 3.X
===

The tiny web server for embedded applications.

### To Read Documentation:

  See wwwdemo/home.htm

### To Build:

    make

Images are built into */bin. The build configuration is saved in */inc/bit.h.

### To Test:

    make test

### To Run:

    ./goahead
